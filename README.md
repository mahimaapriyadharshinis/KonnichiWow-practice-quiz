##  Project Overview

This quiz application allows users to answer multiple-choice questions related to **Japanese language and culture**.  
It provides **real-time feedback**, tracks progress, rewards streaks, and presents a detailed summary at the end.

The project was built as part of a **Frontend Development internship task** with a strong focus on:
- Clean UI
- User experience
- Accessibility
- Code clarity
- Professional standards
